# 文件下载二维码服务

这是一个基于 Flask 的小型服务端程序，用来上传文件并生成下载二维码。手机用微信扫描二维码后，会打开文件下载链接并下载对应文件。

## 功能

- 在网页中选择并上传文件
- 自动生成该文件的下载二维码
- 微信扫码后访问下载链接
- 自动清理 24 小时前上传的文件

## 环境要求

- Windows、macOS 或 Linux
- Python 3.10 及以上

## 安装依赖

```bash
python -m pip install -r requirements.txt
```

## 启动服务

```bash
python app.py
```

启动后浏览器访问：

```text
http://127.0.0.1:5500
```

## 使用方法

1. 打开网页。
2. 把“服务器访问地址”改成你电脑的局域网 IP，例如 `http://192.168.1.10:5500`。
3. 选择一个文件并上传。
4. 页面会生成下载二维码。
5. 用微信扫描二维码，即可打开下载链接。

## 注意事项

- 手机和电脑需要在同一个局域网中。
- 如果电脑开了防火墙，需要放行 `5500` 端口。
- 某些文件类型在微信内可能会先预览或提示跳转浏览器，这是微信自身行为。当前服务已经使用附件下载头，能尽量触发直接下载。
- 上传的文件默认保存在 `uploads` 目录下，并在 24 小时后自动清理。
